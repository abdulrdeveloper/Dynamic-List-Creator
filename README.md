# Dynamic List Creator

A simple and interactive web application that allows users to dynamically create, manage, and delete list items. Built with modern web technologies, this project is perfect for learning and practicing DOM manipulation.

## Features
- Add new items to the list dynamically.
- Mark items as completed with a single click.
- Remove items from the list with a smooth animation.
- Responsive and modern dark-themed UI.

## Technologies Used
- **HTML5**: For the structure of the application.
- **CSS3**: For styling and creating a professional dark theme.
- **JavaScript**: For dynamic functionality and DOM manipulation.
- **FontAwesome**: For icons used in the application.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/abdulrdeveloper/Dynamic-List-Creator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dynamic-list-creator
   ```
3. Open `index.html` in your browser to view the application.

## Usage
1. Enter an item in the input field.
2. Click the "Add Item" button or press `Enter` to add the item to the list.
3. Use the check icon to mark an item as completed.
4. Use the delete icon to remove an item from the list.
